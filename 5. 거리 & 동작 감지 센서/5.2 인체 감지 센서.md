# 인체 감지 센서
## PIR
인체 감지 센서(PIR Sensor: Passive InfraRed Sensor)는 적외선 센서의 일종으로, 일반적으로 인체 적외선 센서라고도 불립니다. 이 센서는 사람이나 동물과 같이 체온을 가진 물체에서 방출되는 적외선을 감지하여 신호를 출력하는 역할을 수행합니다.

설치된 센서는 평상시 주변 환경으로부터의 적외선을 일정하게 감지하고 있습니다. 그러나 사람이 센서 주변으로 접근하거나 동물이 이동하게 되면, 주변 적외선의 분포가 변화하게 됩니다. PIR 센서는 이 변화를 감지하여 출력 핀(OUT 핀)을 통해 디지털 신호(High/Low)를 발생시키게 됩니다. 따라서 마이크로컨트롤러(MCU)나 임베디드 장치에서 손쉽게 감지 신호를 받아 제어 시스템에 활용할 수 있습니다.

### PIR 센서의 동작 원리
- 렌즈(프레넬 렌즈) : 넓은 범위의 적외선을 한 점으로 모아 센서 소자에 전달합니다.
- 센서 소자(Pyroelectric Sensor) : 적외선의 변화량을 감지하여 전기적 신호로 변환합니다.
- 신호 처리 회로 : 변화된 전기 신호를 디지털 신호(High/Low)로 변환합니다.
- 출력 핀(OUT 핀) : 감지 결과를 외부 장치에 전달합니다.

### PIR 센서의 특징
- 저전력 : 대기 전력 소모가 매우 적습니다.
- 저비용 : 구조가 단순하여 가격이 저렴합니다.
- 광범위 감지 : 렌즈 구조에 따라 넓은 감지 범위를 확보할 수 있습니다.
- 다양한 응용 : 보안 시스템, 자동 조명, 가전제품, 무인 감시 장치 등 여러 분야에서 활용됩니다.

## 인체 감지 센서 연결 

| Pin | 연결 대상 | 설명 |
|:---:|:---:|:---|
| 5V | +5V | 전원 |
| GND | GND | 접지 |
| 2 | D | Pir |

## 인체 감지 센서 제어 

```cpp
int pirPin = 2;

void setup() {
  pinMode(pirPin, INPUT); 
  Serial.begin(115200);
}

void loop() {
  int pirValue = digitalRead(pirPin);
  if (pirValue == HIGH) {
    Serial.println("Motion Detected!");
  } else {
    Serial.println("No Motion");
  }
  delay(200);
}
```
